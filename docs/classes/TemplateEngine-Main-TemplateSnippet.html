TemplateSnippet
in package
  • TemplateEngine

TemplateSnippet.php : 24

A template snippet class is used to load a template from a source and apply data from a data provider to it. A template uses tags where data should be placed, a tag looks like this: {{tag_name}}. When the snippet is built all tags that have a corresponding data in the provider will be replaced by that data.
